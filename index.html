后台管理系统

  1. 准备工作，

    1. 安装nrm npm install nrm -g

    2. 下载webpack工具npm install webpack@3.11.0 -g

    3. 下载webpack服务npm install webpack-dev-server@2.9.7 webpack@3.11.0 -D

    4. 全局服务npm i webpack-dev-server@2.9.7 -g

    5. 下载脚手架工具npm install vue-cli -g 下载工具

    6. 通过上面的脚手架工具下载模板 vue init webpack admin

      如果出现下载不了的情况,切换镜像源,并清缓存,卸载重装,webpack非常脆弱

      查看镜像 nrm ls

      选择镜像源 nrm use taobao

      npm cache clean -f 清除缓存
      npm uninstall webpack -g

      npm uninstall webpack-cli -g
